Difflex scenarios is a solution designed specifically for the online stores.

== Description ==

Integration for Difflex e-commerce Service.
Use ready-made scenarios or create your own. Customize widgets and mail messages to increase conversion in the online store.

== Installation ==

1. In admin panel of your WordPress store navigate to the 'Plugins' â†’ 'Add New' section.
2. Type in 'Difflex' in the Search Plugins field and press Enter or download from repository.
3. Press button 'Install Now' on the found plugin.
4. On the 'Difflex settings' page fill in the 'APP Key' field with your Difflex account's 'APP key'.
5. Press 'Save Changes' button.
